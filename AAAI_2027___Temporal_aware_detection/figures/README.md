# Figures

Put figure files here. Allowed formats only: **.pdf, .png, .jpg** (no .eps/.ps/.gif).

- Reference from a section file with, e.g.:
  `\includegraphics[width=0.95\columnwidth]{figures/myplot}`
  (omit the extension; LaTeX picks it up automatically).
- Use **300 dpi** for raster images; prefer vector **.pdf** for plots.
- Crop/resize figures OUTSIDE LaTeX. Do **not** use `clip`/`trim` options.
- Embed all fonts in the figure before importing.
